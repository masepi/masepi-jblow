This is a color theme for VS Code, inspired by Jonathan Blow's emacs color theme.

Core principles:
- Comments are the most important part of the code. They are meant to draw reader's attention to places where something may not work the way they expect, so those places deserve extra attention.
- Function, method, class declarations make it easier to see the structure of the code at a glance.
- Numbers, booleans, and constants are usually code parameters, so it is useful to make them stand out.

In this version of the theme, function calls are not highlighted, although this could be done. This may be added in future versions.
